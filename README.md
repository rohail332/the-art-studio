# 🎨 The  ArtStudio

A modern and responsive multi page website for **The ArtStudio**, showcasing graphic design services, portfolio, pricing and contact information with a clean user experience. 

</div>

## Tech Stack

<div align="center">

[![My Skills](https://skillicons.dev/icons?i=html,css,js)](https://skillicons.dev)

</div>

- **HTML5** - Page structure and content
- **CSS3** - Styling, layout, and responsive design
- **JavaScript** - Interactivity and dynamic content

## Features

- Dynamic counter animation for studio metrics
- Portfolio filtering interactions
- Multi-page layout with services, portfolio, pricing, and contact sections

## Files

- `index.html` - Homepage
- `services.html` - Services page
- `Portfolio.html` - Portfolio gallery
- `Pricing.html` - Pricing and FAQ
- `contact.html` - Contact form and details
- `style.css` - Main stylesheet
- `app.js` - Navigation and animation scripts
- `img/` - Image assets
